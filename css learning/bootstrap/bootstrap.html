element1:

each row = 12 column

element2:

Number of element in each row= 12/number of column.

(ex:12/12=1element in each row or 12/6=2element in each row)

element3:

div class=
"col-xs-12 col-sm-12
col-md-6
col-lg-4 col-xl-4
d-flex justify-content-center"

element4:

means: each row has:
-1 element in xs and small screen size (=12col)
-2 element in md screen size (=6col)
-3 element in lg and xl screen size (=4col)
-and: d-flex justify-content-center ( use to centralize items)
