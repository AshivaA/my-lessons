DOM-page


-when we want to add javascript to html tags we should do that in dom. for example activate a button in html page.
access html tags in DOM:
-we can access the html elements in DOM in 3 form:
(we have 2 ways to select an element by id/class/elements.)

1- by tag name:
a-let example1= document. getElementsByTagName("tagName");
b-let example1= document. querySelector("tagName");

2-by id:
a-let example1= document. getElementById("idName");
b-let example1=document. querySelector("#idName");

3-by className:
a-let example1=document. getElementsByClassName("className");
b-let example1= document. querySelector(".className");

- in document.querySelector, if we want to access all the elements with the same className or id or if we want to access one of the children we should use
let example1= document.querySelectorAll(".className");
==============

-Access an special element:
-if we want to access one of the elements in li element, we should access it by index number.
ex:let lis= document.getElementsByTagName("li");
console.log(lis) =>HTMLCollection(4) [li.li-ex, li.li-ex, li.li-ex, li.li-ex]
console.log(lis[1]);=> second child( into li tags)

we can access an element in dom from (parents to child) or (child to parents)

-(parents to child): to select an special tag we should start counting from parents tag to children tags.
let forthChild= document.querySelectorAll(".div-ex li.li-ex")[3];
console.log(forthChild);=> forth-child(into li tags)

explanation: (".div-ex li.li-ex")[3]=>
- .div-ex=(div className)
- li.li-ex=(li.li className)
- [3]=(index of forth child)
Traverse in DOM (parents to children):

-access parent tag:
let ulParents= document.querySelector("ul");
console.log(ulParents);

-access special children content:
let Children= ulParents.children;
console.log(Children[1].textContent);=> second-child

-access first/last child/number of children:
let firstChild= ulParents.firstElementChild;
let lastChild= ulParents.lastElementChild;
let numberChildren= ulParents.childElementCount;

console.log(firstChild);=> first-child(inside li tag)
console.log(lastChild);=> forth-child(inside li tag)
console.log(numberChildren); => 4

local storage:
(local storage use to save data in browser.)
important:
-we can only save the data in local storage in string.
-access the local storage in inspect:
Application-> local storage-> open it -> go to http:---
=======
-making local storage:
1-set key, value to access data:
localStorage.setItem("myKey", "myValue");

2-store data in a variable and access the values by the key:
let storData= localStorage.getItem("myKey");

console.log(storData);
=======
-remove special data from local storage:
localStorage.setItem("shiva","khandel");
localStorage.setItem("ahoora","barana");

let shivaName= localStorage.getItem("shiva");

localStorage.removeItem("shiva");=> shiva khandel will remove.

-remove all data from local storage:
localStorage.clear();=> all local storage data will remove.
=======
-Saving an array in local storage:
(we should only put the key in string and not arrayName)

1-create an array:
let myArray= [1,2,3,4];

2-create "key",value(arrayName) :
localStorage.setItem("arr",myArray);

3-store array in a variable:
let storArray= localStorage.getItem("arr").split(",");

console.log(storArray);=> (4) ['1', '2', '3', '4']
Application=> 1,2,3,4 (this is not a string)
===========
-Saving an object in local storage:
to store an object we should:

first: convert object to string by:
localStorage.setItem("key", JSON.stringify(objectName))

then: convert string to object by:
JSON.parse(localStorage.getItem("key"))

1-make an object:
let myObject= {
name: "shiva",
age: 38
}

2-convert object to string:
localStorage.setItem("obj", JSON.stringify(myObject));

3-convert string to object and store object:
let storeObject= JSON.parse(localStorage.getItem("obj"));

console.log(storeObject);=> {name: 'shiva', age: 38}
Application=> {"name":"shiva","age":38} (shows as a string)
