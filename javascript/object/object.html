constructor:
-when we want to have several objects that do the same things we use constructor.
- constructor is a function or class that can create and return a new empty object.
-important: the first letter of constructor(function name) should start with capital letter.

-in constructor we should:

1-make object properties(key:value):
this.key= parameter(as a key value);
(that will makes(properties) key : value inside the new object)
-this=object name
that means when ever we add a new argument to specific object this will change to that specific object.
(ex: this.firstName = person1.firstName =>shiva )
(ex: this.firstName = person2.firstName =>amin )

2- make new object:
let objectName= new constructorName(argument);=>( this is new object)
ex: let person1= new Bio("shiva","khandel");=>new object

===============

-function-constructor:

-example:
function Bio(fName,lName) {
this.firstName= fName;
this.lastName= lName;
}

new object: let person1= new Bio("shiva","khandel");
new object:let person2= new Bio("amin","rahmat");

method: Bio.prototype.fullName= function () {
return `${this.firstName} ${this.lastName}`;
}

console.log(person1);=>Bio {firstName: 'shiva', lastName: 'khandel'}

console.log(person2);=>Bio {firstName: 'amin', lastName: 'rahmat'}

console.log(person1.fullName());=>shiva khandel
console.log(person2.fullName());=>amin rahmat

-class-constructor:

-example:
class Person{

constructor(fName,lName){
this.firstName= fName;
this.lastName= lName;
}

method: fullName(){
return `${this.firstName} ${this.lastName}`;
}
}

new object: let shiva= new Person("shiva","khandel");
new object: let amin= new Person("amin","rahmat");

console.log(shiva);=>Person {firstName: 'shiva', lastName: 'khandel'}
console.log(amin);=>Person {firstName: 'amin', lastName: 'rahmat'}
console.log(shiva.fullName());=>shiva khandel
console.log(amin.fullName());=>amin rahmat
