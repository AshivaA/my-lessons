Array-page



-Array: is a collection of values. they can be the same or different datatype. but it is better to use only one of datatype in an array.
ex: let number=[1,4,90]; or let collection=[4,"text",true];

example:
this is our array shows in console: (3) [2,"text",true]
-if we consider an array as a hotel, the values inside in, should assume as the guests of each hotel rooms or index so:
-(3) is: the length of an array (the hotel has 3 rooms)
-(0:2) means: (the index of 2 is 0) or (2 is in room number 0)
-(1:'text') means: (the index of 'text' is 1) or ('text' is in room number 1)
-(2:true) means: (the index of true is 2) or (true is in room number 2)
(we have index only in array)

-access the special value:
to access the special array value we need index number ex:
console.log(dataTypArray [0]); = 2
console.log(dataTypArray [1]); = 'text'
console.log(dataTypArray [2]); = true

-add or replace a value in an array:
we only need to change the index:
add:
(dataTypArray [3] = 77;) in console we have: (4) [2,"text",true,77]

replace:
(dataTypArray [2] = false;) in console we have: (4) [2,"text",false,77]

normal array method:

let numbers=[0,1,2,3,40,50,60,7,8,9,10];
let result;

result= numbers.length;(length is a property so don't need () )
result= numbers.indexOf(5);(shows the item in this index)

----------

-unshift/shift
result= numbers.unshift(-1);( add a number from the start of an array)
result= numbers.shift();( remove a number from the start of an array. we don't need to set any number)

----------

-push/pop
result= numbers.push(11);( add a number from the end of an array) result= numbers.pop();( remove a number from the end of an array. we don't need to set any number)

----------

-replace/remove new number
result= numbers.splice(2, 0, 80);(replace new number in special index)
(2=index[2], 0=keep the new number in the same index, 80=the new number)
result= numbers.splice(5, 1);( remove a number from special index)
(5 = when rich index 5, 1= then remove the item in index5)

-remove a series of numbers from an index to another index:
result= numbers.splice(7, 2);
(7 = when rich index 7, 2= then remove the item in index7, and the next one )

-sort ():will change the original array.
let Numbers= [3,91,7,67,0,52,40];

-sort numbers in order:
SortNumOrder=Numbers.sort(function(x,y){
return x-y;(sort numbers left to right)
});
=>[0,3,7,40,52,67,91]

-sort numbers in revers:
SortNumRevers= Numbers.sort(function(x,y){
return y-x;(sort numbers right to left)
});
=>[ 91,67,52,40,7,3,0]

important:
-we can not sort the array numbers by [sortNumbers= Numbers.sort();] because the javascript consider only the first number to sort the numbers so the 67 consider as 6/ or 91 consider as 9.
-the result is:[0,3,40,52,67,7,91]

let Numbers= [3,91,7,67,0,52,40];
let sortNumbers;
sortNumbers= Numbers.sort();=>[0,3,40,52,67,7,91]

-filter ():can make new array to save results so do not change the original array.
let Numbers= [3,91,7,67,0,52,-1,-9,40];

filterResult= Numbers.filter(function (Number(shows array items)) {
return Number < 0;
})

=>[-1, -9]

how filter() works:
-in first loop:
1-compare first Numbers item or index[0]< 0 so the first loop is 3<0
2-if true then save it in filterResult if false then ignore it.
and this is the same way to check all items in an array.

--------------

-map(): can make new array to save results so do not change the original array.

let Numbers= [-1,-9,0,3,40,52,67,7,91]

mapResult=Numbers.map(function(Number) {
return Number + 1
})
mapResult= [0, -8, 1, 4, 41, 53, 68, 8, 92]
