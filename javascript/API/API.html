-Synchronous:
meaning: start now, use it at the same time.
-in java script codes(statement) executes line by line and that means code is Synchronous.

-description:
statements: In JavaScript, statements are instructions that are executed by the browser or Node. js
ex: variables, functions, if, loop, console.log()
---------
-Asynchronous:
meaning: start now, use it when it finished.
-in java script some codes(statement) need mor time to finish(ex:3 second) and then execute, that means code is Asynchronous.

-so if we want to execute a code without wasting the time to execute we need to use Asynchronous function.

ex: setTimeout(function, time)
is an Asynchronous function that means execute this function after passing this special time.

setTimeout(() => {
console.log("shows HELLO after three second");
}, 3000);
-------
-Web API:
the time that an Asynchronous code needs to finish and then execute happen in another place(web page) outside of your code, called API.
------
-Callback function:
meaning: when that time is finished then the result shows as a callback function.

Manage Asynchronous code:
-callback function and promise object use to manage Asynchronous code which means:
-in normal calling Asynchronous code, the Asynchronous code will execute randomly if we do not manage them. so to manage them in what order and when the Asynchronous code to execute we need to use callback function or promise object.

callback function:
-when we use a function as an argument in another function, the argument function is callback function.

comparison:

ex 1:(normal calling function with setTimeout)
function Func1(){
setTimeout(function (){
console.log ("the first function execute after second function");
},1000);
}

function Func2(){
console.log ("the second function execute first ");
}

Func1();
Func2();

ex 2:(callback function with setTimeout)
function Func1(Func){
setTimeout(function (){
console.log ("the first function execute first.");
Func();
},3000);
}

function Func2(){
console.log ("the second function execute second.");
}

Func1(Func2)
(Func2)=> is a callback function.

callback hell:
-when we use several call back function into one function it calls callback hell.
