Function-page


-function-syntax:

function funName(parameter1,parameter2){
code to be executed
}
funName(arguments);

-rest parameter syntax:(should be in parameters)

function sum (...rest parameter) {
let result = 0;
for (let arguments of rest parameter){
result += arguments;
}
return result;
}

console.log(sum(arguments));

-spread syntax:(should be in arguments as an array)

let spreadArray= [2,3];
function Sum(...restParameter) {
let result= 0;
for (const arguments of restParameter) {
result+= arguments;
result= [1,...spreadArray,4];
}
return result;
}

console.log(Sum(1,...spreadArray,5));=>11
console.log(Sum(...spreadArray));=>[1, 2, 3, 4]

-Return word in function:
to show the result on console we should use the word return in function.

example:
function square(PNumber) {
return PNumber * PNumber;
}

console.log(square(3)); => 9
console.log(square(5)); => 25
console.log(square(4)); => 16

important:
-we should check if our function is working or not by testing it to show a text in console.
ex: console.log("this function is working")

-rest:(should be in parameter)
- will store the rest arguments in rest parameter(...varName) as an array, when the number of arguments is more than parameter
-(...varName) should be the latest parameter.

-rest meaning:
function bio(fName, lName,...otherInfo) {

return otherInfo;=>['web dvp', 'artist']
return [fName, lName,...otherInfo]; =>['shiva', 'khandel', 'web dvp', 'artist']
}

console.log(bio("shiva", "khandel", "web dvp","artist" ));
...otherInfo = ...varName (is an array that store extra argument)

-rest ex:
function sum(a,b,...restParameter) {
let result= 0; ( result start counting from 0)

for (let arguments of restParameter) {
result+= arguments;
}
return result;
}

console.log(sum(1,3,2,5));=>6
a=1
b=3
2+4=...restParameter

-spread:(should be in argument)

will expands array items in(...varName) in argument
-we can only use it in an object literal (like array or string)
-( ...spreadArray) can be at first or in the middle of arguments

let Array1= [1,2,3,4,5];
