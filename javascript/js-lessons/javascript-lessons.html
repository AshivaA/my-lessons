Assignment:

-we have 2 kind of assignment normal and deep.
example:
let person={name:"shiva"}
let anotherPerson;

-normal assignment is:( anotherPerson=person; )
when we assign 2 variables in normal way, and chang one of them, then all the variable value will chang to the new one.

anotherPerson=person;
anotherPerson.name ="amin";

console.log(person);=> {name:"amin"}
console.log(anotherPerson);=> {name:"amin"}

-deep assignment is: [ anotherPerson=Object.assign ({}, person); ]
when we set [Object.assign ({}, person)] for second variable, then the new value stack on new object or array.

anotherPerson= Object.assign({}, person);
anotherPerson.name="amin";

console.log(person);=> {name:"shiva"}
console.log(anotherPerson);=> {name:"amin"}

exactly the same for array:
let DpNumber=[1,8,46,9];
let newDpNumber;

newDpNumber= Object.assign([], DpNumber);
newDpNumber[2]=100;
console.log(DpNumber);=> (4) [1,8,46,9]
console.log(newDpNumber); => (4) [1,8,100,9]
